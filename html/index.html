¿Qué es la usabilidad web?

Se conoce como Usabilidad Web a la facilidad que provee una página web en su uso. Tanto la ejecución técnica como la facilidad en el uso se encuentran estrechamente ligados con la apariencia y el diseño del sitio web.

Importancia de la usabilidad web

Al centrarse en otorgar facilidad en el uso de una página, la importancia de la usabilidad web radica en que esta ayuda a que los usuarios puedan utilizar la página web ´varias veces´ sin complicaciones. Esto ayuda a generar un tráfico más elevado hacia la página, lo cual beneficia el mercadeo del sitio así como su posicionamiento en los buscadores de Internet.

Principios

Principios de usabilidad web
Principio Definición
Disponibilidad Esto significa qué tan fácil un sitio web se puede acceder. La establidad de la infraestructura que provee el sitio web en Internet puede afectar la disponibilidad.
Claridad Esto representa qué tan alineado está el sitio web con el servicio que ofrece. En otras palabras, los usuarios esperan que el sitio se alinee a sus objetivos. Por ejemplo: un usuario espera que un sitio cuyo objetivo sea la compra de equipo de ferretería le permita realizar la compra de forma sencilla y efectiva.
Reconocimiento Los usuarios tomarán unos segundos para poder obtener una imagen visual del sitio web y tratarán de asimilarlo con otros sitios web que proveen ese servicio. Aquí la facilidad de uso y la apariencia del sitio son determinantes.
Credibilidad Si el sitio web no tiene un objetivo o un propósito específico, los usuarios no van a confiar en él. Esto afectará negativamente la confiabilidad en este.
Relevancia Aquí entra en juego el público objetivo, ya que las personas se sentirán más atraídas a consumir servicios que se adapten a sus necesidades.

Ejemplos

  • Disponibilidad: IBM es uno de los proveedores de 'hosteo' de servicios web más conocidos, y provee el servicio 'IBM® HTTP Server for i'. Este servicio provee una alta disponibilidad y escalabilidad de los servidores web en Internet.
  • Claridad: La página web de la ferretería EPA provee en su página web un catálogo de herramientas/equipo disponible para comprar con un click. Esto permite al usuario conocer de inmediato cuál es el objetivo de la página.
  • Reconocimiento: El sitio web de Amazon Prime Video ofrece una interfaz similar a la de otros sitios web de streaming, como Netflix y Disney Plus. Esto permite a los usuarios rápidamente asociar el sitio web con películas, series, documentales, etc.
  • Credibilidad: El sitio web GoFundMe.com muestra en detalle las causas para donar y el objetivo de la donación. También muestra información de los donantes, así como cuánto se ha recaudado del total estipulado
  • Relevancia: El sitio web de GNC provee información inmediatamente sobre multivitamínicos y proteínas para las personas que gustan de cuidar su salud una vez que se accede a la página web.

    • NOTA: todos los ejemplos provistos arriba cumplen con los principios anteriormente mencionados.
