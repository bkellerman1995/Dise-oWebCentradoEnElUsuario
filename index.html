¿Qué es diseño web centrado en el usuario (DCU)?

Se conoce como Diseño Web Centrado en el usuario (DCU) a una metodología de diseño de una página web enfocada en el usuario. Se prevee cómo va a ser la experiencia del usuario en cada etapa del diseño. Esto incluye tanto la conceptualización como la implementación (materialización).

Importancia del DCU

Debido a que el enfoque del DCU se centra en la experiencia del usuario al consumir el sitio web que estamos diseñando, durante la etapa de diseño podemos entender qué está funcionando bien, qué no y por qué. El usuario final se vuelve un aliado productivo en la creación del sitio web y nos sirve como alerta para entender qué podemos mejorar o qué valor agregado puede tener al final nuestro sitio web para agradar y facilitar el uso al público general. Un DCU bien estructurado ayuda a prevenir fallas de diseño y rechazo al involucrar usuarios de prueba y generacionales que activamente van a proveer retroalimentación temprana.

Pasos o etapas del DCU

DCU paso a paso
Paso Definición
Investigación Se analiza el tipo de usuario que va a hacer uso del sitio web: sus características. Debe haber un entendimiento del entorno social y las tareas que realiza el usuario en su día a día.
Concepto Se realiza un estudio de factibilidad de las soluciones que se implementarán en el sitio web y se verifican (prueban) si estas se adaptan a los usuarios objetivo. Además, se determina si financieramente son viables.
Diseño Comprende desde un concepto aproximado al diseño completo del sitio web. Este abarca un proceso de aprendizaje donde se identifican oportunidades, nuevas ideas surgen y mediante prueba y error se perfeccionan las soluciones.
Evaluación Este paso es crucial ya que se obtiene la retroalimentación en tiempo real de una muestra de los usuarios que conformarán el grupo de usuarios finales. Este proceso se encuentra ligado con las pruebas de calidad del software, que aunque parecida, no es exactamente lo mismo.
Lanzamiento Luego de que el sitio web se encuentre activo, se debe realizar un seguimiento activo para comprobar si el resultado final es el deseado para el usuario. Aquí, la evaluación de variables cualitativas junto con el desempeño del sitio web se tornan la rutina post-lanzamiento.

Beneficios

  1. Para el usuario

    • Mejora de la usabilidad: El sitio web que sigue los principios de DCU ayuda a evitar errores no deseados por parte del usuario.
    • Experiencia positiva: El sitio al cumplir su objetivo de satisfacer una necesidad del usuario final provee resultados satisfactorios.
    • Aumento de eficiencia: Un diseño bien implementado mejora la eficiencia en su uso. La productividad también es afectada positivamente.
    • Mejora de accesibilidad: Un sitio web que sigue los principios de DCU cumple el requisito de accesibilidad por naturaleza.

  2. Para la empresa

    • Reducir riesgos y costes: Al trabajar directamente con el usuario durante el proceso de diseño, muchos riesgos pueden llegar a ser aplacados con anterioridad. Esto evita costes elevados en producción por errores no identificados en el desarrollo, lo cual DCU ayuda a
    • Fidelizar a los usuarios: Si el sitio web satisface necesidades del mundo real, es muy probable que se cree un nicho de clientes regulares que frecuenten el sitio web.
    • Mejorar la imagen de marca: Los sitios web centrados en el usuario tienen un alto grado de aceptación. Esto ayuda con la proyección de imagen de marca mejorada.
    • Aumentar ventaja competitiva: Optar por la investigación de los usuarios finales ayuda a ampliar ventajas en los mercados competitivos. Una conexión directa con potenciales clientes aporta significativamente en el camino del éxito de un servicio web.

Ejemplos

  • Microsoft: Microsoft ha cambiado el enfoque de la interacción con sus usuarios y sus sitios web están diseñados para adaptarse al contexto del cliente final. Por ejemplo, en su sitio office.com ofrece de entrada herramientas destinadas a apoyar al usuario de manera fácil y efectiva con sus tareas profesionales o de estudio.
  • YouTube: YouTube ofrece un sitio web que se adapta al contenido consumido por el usuario. Esto garantiza que el contenido que se ofrece o recomienda sea de agrado al usuario o afin a sus gustos. Esto le ha ayudado a recibir millones de usuarios diariamente por muchos años.
  • Waze: Aparte de ofrecer un sistema de GPS y navegación estándar para los usuarios, Waze muestra de antemano los estados de carretera (libre, congestionado), presencia de policías, derrumbes, etc. Se crea un perfil de navegación ajustado al usuario el cual se va adaptando basado en el uso que el usuario le dé.

    • NOTA: todos los ejemplos provistos arriba cumplen con los principios de DCU anteriormente mencionados.
